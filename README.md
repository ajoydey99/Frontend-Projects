# 📁 Frontend Fundamentals Project

This repository contains a collection of folders that cover the core building blocks of frontend web development. Each folder focuses on a specific area to help strengthen understanding and practical skills in HTML, CSS, and design principles.

---

## 📂 Folder Structure

### 1. HTML Fundamentals

This folder includes the basics of HTML (HyperText Markup Language). It covers:

* Basic HTML structure
* Common tags and elements
* Forms and input fields
* Semantic HTML

---

### 2. CSS Fundamentals

This section focuses on core CSS (Cascading Style Sheets) concepts:

* Selectors and specificity
* Colors, fonts, and text styling
* Box model
* Display properties

---

### 3. CSS Layouts

This folder demonstrates modern layout techniques:

* Flexbox
* CSS Grid
* Positioning (relative, absolute, fixed)
* Responsive design basics

---

### 4. Design

This section contains design-related work and concepts:

* UI design practice
* Layout planning
* Color theory basics
* Visual hierarchy

---

## 🚀 Purpose of This Repository

The goal of this project is to build a strong foundation in frontend development by organizing learning materials and practical examples in a structured way.
